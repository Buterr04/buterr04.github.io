# Flutter
> Flutter是Google开发的一款软件包，能够构建高性能、高保真的移动应用。在更新之后可以支持Web、桌面应用和Android以及iOS设备，能够实现跨平台开发。

## 安装Flutter
1. 下载Flutter SDK：访问[Flutter官网](https://flutter.dev/docs/get-started/install)下载适合你操作系统的Flutter SDK。
2. 解压缩下载的文件到你希望安装Flutter的目录。
3. 将Flutter的`bin`目录添加到你的系统环境变量中，以便在终端或命令行中可以直接使用`flutter`命令。
4. 在终端或命令行中运行`flutter doctor`命令，检查Flutter的安装状态和依赖项。根据提示安装缺失的依赖项。
